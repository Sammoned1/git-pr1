# Чат-бот

Это полнофункциональное приложение чат-бота с использованием современного стека:

- **Frontend:** React + TypeScript
- **Backend:** Node.js + Express + TypeScript

## Серверная часть

```sh
cd server
```

```sh
npm i
```

```sh
npm run dev
```

## Клиентская часть

```sh
cd client
```

```sh
npm i
```

```sh
npm start
```
